# jsDetectClicks

Research and development: Project4Dimensions

## Why jsDetectClicks?

jsDetectClicks is a JavaScript algorithm to determine button clicks; 
it could be used to invoke alternate procedures.

See [`detect-clicks.js`](detect-clicks.js).
